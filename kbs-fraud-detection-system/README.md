# ?? Fraud Detection System (KBS)

## ?? Overview
This project is a **Knowledge-Based System (KBS)** that detects potentially fraudulent transactions using rule-based reasoning.

---

## ?? How It Works
The system takes:
- Transaction amount  
- Location (usual/unusual)  
- Time (day/night)  

It applies IF–THEN rules to determine fraud risk.

---

## ?? Features
- Rule-based decision system  
- Fraud risk classification (Low, Medium, High)  
- Explanation of decisions  

---

## ?? Example

Input:
Amount: 150000  
Location: unusual  
Time: night  

Output:
HIGH RISK: Fraud Detected  
Reason: Large transaction in unusual location  

---

## ?? KBS Concepts Used
- Knowledge Base (rules)  
- Inference Engine  
- Rule-Based System  
- Explanation Facility
